# npm scripts

- inside package.json, within scripts object add cli commands that you want to run.
- ex : "Scripts: {
    "start": "parcel index.html",
    "build": "parcel build index.html"
  }

- to run the script added, run 
- --> npm run start / npm start 
- --> npm run  build
